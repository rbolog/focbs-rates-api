# API worker

## Dev

### Setup

`npm install wrangler@latest`

`cp wrangler.toml.sample wrangler.toml`

#### dependencies

`npm install luxon`

`npm install --save-dev @types/luxon`

## run dev

Cron worker should be deployed and kv store should have data

`npx wrangler dev --remote`

## Setup and deploy

`cd api`
